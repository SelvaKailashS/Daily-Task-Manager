# SUPABASE SETUP GUIDE

## Step 1 — Create Supabase Account
1. Go to https://supabase.com
2. Click "Start your project" → Sign up free
3. Click "New Project"
4. Fill in:
   - Name: Stoners
   - Database Password: (create a strong password — SAVE IT)
   - Region: pick closest to you
5. Click "Create new project" → wait ~2 minutes

## Step 2 — Get Database URL
1. In your Supabase project → click "Settings" (gear icon)
2. Click "Database" in the left menu
3. Scroll down to "Connection string"
4. Select "URI" tab
5. Copy the connection string — looks like:
   postgresql://postgres:[YOUR-PASSWORD]@db.xxxxxxxxxxxx.supabase.co:5432/postgres
6. Replace [YOUR-PASSWORD] with the password you created

## Step 3 — Add to Render
1. Go to your Render dashboard
2. Click your "stoners-app" service
3. Click "Environment" tab
4. Click "Add Environment Variable"
5. Key:   DATABASE_URL
   Value: (paste your Supabase connection string)
6. Click "Save Changes"
7. Render will redeploy automatically

## Step 4 — Done!
Your app now uses Supabase PostgreSQL.
Data persists forever — no more resets on redeploy!

## Local Development
When running locally (python app.py), it still uses SQLite automatically.
Only the online Render version uses Supabase.
No changes needed for local use.
